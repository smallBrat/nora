# Contact

> Get in touch about Nora — bug reports, security issues, partnership questions, or anything that doesn't fit the public Issues tracker.

Most questions and bug reports are handled best in the open. Use the right channel for the right kind of conversation.

## GitHub

For anything that benefits from a public, searchable record:

- **Bug reports and feature requests:** [github.com/solomon2773/nora/issues](https://github.com/solomon2773/nora/issues)
- **Open-ended discussion, "is this the right approach?", show-and-tell:** [github.com/solomon2773/nora/discussions](https://github.com/solomon2773/nora/discussions)
- **Pull requests:** fork, branch, PR. See [`CONTRIBUTING.md`](https://github.com/solomon2773/nora/blob/master/CONTRIBUTING.md) before deeper repo work.

## Security

Please do **not** open a public issue for security-sensitive reports. See [`SECURITY.md`](https://github.com/solomon2773/nora/blob/master/SECURITY.md) for the disclosure process.
