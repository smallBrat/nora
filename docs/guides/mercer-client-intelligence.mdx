# Set up Mercer, the client-intelligence agent

> Install the Client Intelligence & Sales Momentum template, connect one channel, optionally connect a CRM, and give Mercer your sales context. Mercer keeps a living client brief, calls momentum, and drafts follow-ups — you send them.

Mercer is a relationship-tracking agent built as an Agent Hub template. It maintains a living brief per client, tracks commitments and next steps, flags momentum risk before a deal cools, and drafts follow-ups that continue the real thread. **Mercer drafts, you send**, and never invents client intent.

<Tip>
  **Mercer** is just the default name. On first run the agent asks what you'd like to call it —
  rename it, or keep "Mercer."
</Tip>

## Before you install

- A running Nora instance with dashboard access.
- Your offer, sales cycle, ICP, and pipeline stages (if you use them).

## Step 1 — Install from Agent Hub

Open **Agent Hub**, find **Client Intelligence & Sales Momentum Claw**, and click **Install**. See [Publish and install Agent Hub templates](/guides/agent-hub).

## Step 2 — Connect a channel

Mercer needs **one** channel to send you briefs and follow-up drafts (Channels tab).

<Steps>
  <Step title="Create a WhatsApp Cloud API app">
    At [developers.facebook.com](https://developers.facebook.com), add the **WhatsApp** product.
  </Step>
  <Step title="Copy credentials">Copy the **Phone Number ID** and an **Access Token**.</Step>
  <Step title="Add the channel">
    Agent → **Channels** tab → add **WhatsApp** → paste the values.
  </Step>
</Steps>

Or use Telegram / Slack / Discord / email — see [Configure channels](/guides/channels).

## Step 4 — Run the bootstrap

Send any first message. Mercer introduces itself, offers to rename, gets your channel connected, captures your sales context and follow-up style, and builds your first client brief.

## What Mercer will and won't do

- **Draft-only.** Mercer never sends, and never fabricates facts not in the record.
- **Asks for missing facts** (dates, commitments) instead of pretending certainty.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Mercer says a CRM isn't connected, but I added it">
    Mercer reads `integrations/NORA_INTEGRATIONS.md`, regenerated on sync. Re-check the
    **Integrations** tab.
  </Accordion>
  <Accordion title="Momentum call seems off">
    Momentum needs concrete dates and commitments — give Mercer the latest interaction with timing
    and it will recalibrate.
  </Accordion>
</AccordionGroup>
