# Google Cloud Console screenshots - GKE setup

These are real Google Cloud Console screenshots used by the GKE provisioner-backend docs. Google Cloud's current GKE docs did not publish matching dashboard captures for each Nora doc step, so these use public GKE walkthrough captures of the same console surfaces.

- `cluster-details.png` - Google Cloud Console Kubernetes Engine clusters view from the TechWithHuz GKE walkthrough image `create-cluster-3.png`.
- `connect-dialog.png` - Google Cloud Console GKE Connect dialog from the LinuxTechi GKE walkthrough image `Command-line-access-k8s-gcp.png`.
- `networking.png` - Google Cloud Console Autopilot cluster review page from the TechWithHuz GKE walkthrough image `create-cluster-1.png`.

Do not replace these with generated art or hand-drawn stand-ins. Refresh with real Google Cloud Console captures when the GKE UI changes.
